# 📊 Data Analytics Project

## 🔍 Overview

This project demonstrates a complete data analytics workflow, starting from raw data to actionable insights. It covers data loading, cleaning, exploratory data analysis (EDA), SQL querying, and data visualization through an interactive dashboard. The final results are presented in both a report and a presentation.

The goal is to showcase practical skills in data analysis and business intelligence using industry-standard tools.

## ⚙️ Project Steps

### 1. Data Loading

* Imported dataset using Python (Pandas)
* Initial inspection of data structure and types

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Corrected data types
* Standardized column names

### 3. Exploratory Data Analysis (EDA)

* Statistical summaries
* Data visualization (charts, distributions, correlations)
* Identification of patterns and trends

### 4. SQL Analysis (PostgreSQL)

* Loaded cleaned data into PostgreSQL
* Wrote SQL queries to extract insights

  * Aggregations (SUM, AVG, COUNT)
  * Joins and filtering
  * Grouping and sorting

### 5. Dashboard Creation

* Built an interactive dashboard in Power BI
* Included key KPIs and visualizations
* Enabled filtering and drill-down features

### 6. Reporting & Presentation

* Created a detailed analytical report
* Designed a professional presentation using Gamma
* Highlighted key findings and recommendations

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run Python Analysis

* Open the Jupyter Notebook or Python script
* Execute all cells to perform EDA and data cleaning

### 4. Setup PostgreSQL

* Create a database
* Update connection credentials in the script
* Run SQL scripts to load and query data

### 5. Open Power BI Dashboard

* Open the `.pbix` file in Power BI Desktop
* Refresh data if needed
